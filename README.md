# Team Collaboration System

This project implements a role-based file access and tracking system in Python.

## Modules
- login.py – User authentication (3 attempts)
- authorization.py – Role-based file access control
- action_execution.py – File editing, backup, and audit logging

## Features
- Role-based access (Admin, Instructor, Student)
- Closed project restriction
- Backup and audit tracking
- Group-based file permissions
